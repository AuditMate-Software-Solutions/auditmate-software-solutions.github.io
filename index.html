About

AuditMate is a portable system tool that collects and records the exact state of your computer at a given moment.

  • Users and administrator accounts
  • Running services
  • Installed packages
  • Open ports
  • Firewall state
  • System drift detection

Works fully offline, requires no installation, and produces clean audit reports.
