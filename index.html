AuditMate — Portable System Audit Evidence Tool

AuditMate is a portable, offline system audit tool that produces deterministic, verifiable snapshots of system state. It captures factual system evidence that can be reviewed by humans, compared over time, and used directly for audits, incident response, and system trust.

Open Source Core

AuditMate’s core scanning and baseline functionality is fully open source. The core can be run offline, inspected, verified, and customized without restrictions.

Advanced reporting, export formats, and premium automation features are part of the paid version and remain closed source to ensure long-term sustainability.

Key Capabilities

System Evidence Collection

  • Hostname, operating system, kernel version, and uptime
  • Administrator and regular user accounts
  • Running services
  • Installed packages
  • Listening network ports
  • Firewall state

Baseline & Change Tracking

  • First run automatically creates a baseline snapshot
  • Subsequent runs compare against the baseline
  • All changes are shown as explicit additions or removals
  • No background agents or continuous monitoring

Audit & Compliance Ready

  • Produces human-readable, deterministic evidence
  • Designed to cover the majority of SOC 2 CC6 & CC7 technical checks
  • Output can be archived and handed directly to auditors

Automation Friendly

  • Designed for cron, Task Scheduler, CI pipelines, and offline execution
  • Consistent output suitable for scripting and verification
