AuditMate — Portable System Audit Evidence Tool

AuditMate is a portable, offline system audit tool that produces deterministic, verifiable snapshots of system state. It captures factual system evidence that can be reviewed by humans, compared over time, and used directly for audits, incident response, and system trust.

Why AuditMate?

  • Automated Monitoring: Continuous visibility without manual checks.
  • Security & Compliance: Detect critical changes like new admin users, open ports, or disabled firewalls.
  • Historical Tracking: Save baselines and compare system state over time.
  • Audit Ready: Covers 80–90% of SOC 2 CC6 & CC7 technical checks.
  • Lightweight & Offline: Runs anywhere with no internet required.

Product Tiers

Free Tier — $0

Designed for administrators or individuals who need reliable core system visibility.

  • System info: hostname, OS, kernel, uptime
  • User accounts: administrators and regular users
  • Running services
  • Installed packages
  • Open ports
  • Firewall state
  • Pending system updates
  • Baseline comparisons and critical alerts
  • Terminal summary and exit codes
  • Fully offline, open source, verifiable

Ideal for personal systems or small teams that need visibility without overhead.

Paid / Premium Tier — $25 / month or $250 / year

Includes everything in the Free tier, plus advanced reporting and audit-ready features for professional environments.

  • Full historical tracking and trend analysis
  • JSON / CSV / HTML report exports
  • Detailed human-readable audit reports
  • Multi-system aggregation
  • Notifications (email, Slack, webhooks)
  • Premium licensing for multiple devices
  • Closed source with secure updates

Ideal for companies preparing for audits or managing multiple systems.

How It Works

  1. Collect: AuditMate gathers current system state.
  2. Baseline: Save a snapshot for future comparison.
  3. Diff: Identify what changed.
  4. Report: Terminal summary (Free) or detailed reports (Premium).
  5. Automate: Schedule via cron, Task Scheduler, or CI.
