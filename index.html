About

AuditMate is a lightweight system auditing tool that snapshots and compares the state of a machine locally.

  • Detect users, services, packages, and open ports
  • Compare system changes over time (baseline vs current state)
  • Works completely offline
  • No installation required (single binary)
  • Produces clean terminal and JSON reports

Designed for audits, debugging, and system verification without cloud dependencies.

Veteran-owned and operated
